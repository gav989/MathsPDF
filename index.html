Click to download a zip file containing sets of Maths questions targetted at KS3 and GCSE students. The file contains the following folders:
  • Board - Sets of questions on specific topics/skills intended to be projected onto a board
  • Exam Questions - Exam questions with the numbers changed, based on questions that have appeared in OCR exam papers
  • GCSE Gym - Inspired by the popular Numeracy Ninjas programme, this is a (work in progress) programme for KS4 students with a focus on key skills for the GCSE, content memorisation and exam questions (from past OCR papers)
  • Homeworks - Printable sets of questions designed to be cut up for homeworks (one column per student)
  • Insight - Separate documents for questions and answers, for teachers who set homework online. Formatted to fill a mobile phone screen.
  • Starters - Starters that I use with my classes
  • Times Tables Programme - Contains printable sheets of questions intended to develop times tables skills
  • Worksheets - Printable sets of questions fitted to a single A4 page
To go fullscreen, press ctrl+l (or f5, or f11). Then use the mouse buttons to go forwards or backwards through the slides.

The questions are regenerated and uploaded to this page each day.
